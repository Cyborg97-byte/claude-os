## 🧠 Learnings über Claude

| Learning | Details |
|----------|---------|
| Copy-paste-ready einfordern | Steht jetzt in Projektanweisungen — Claude liefert es automatisch |
| Screenshots > Beschreibungen | Claude sieht direkt was Sache ist — spart 3-4 Nachrichten |
| Notion MCP direkt nutzbar | Claude kann Seiten erstellen, updaten, Datenbanken abfragen |
| Chrome Extension: Domain-Freigabe | Jede Domain muss einzeln freigegeben werden (Settings → Allowed Sites) |
| notion.so ≠ notion.site | Eingeloggte Version (notion.so) für Extension, public (notion.site) separat freischalten |
| Token nie im Chat teilen | GitHub API aus Container nicht erreichbar — Token sofort löschen wenn geteilt |
