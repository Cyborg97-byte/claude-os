# Prompt: Produktbilder als PNG

## Verwendung
Wenn Claude Produktbilder erstellen soll.

## Prompt
```
Erstelle mir [ANZAHL] Produktbilder als PNG (2000x2000px) für [PRODUKT].

Stil: Dunkler Hintergrund (#0F0F14), grüner Akzent (#10B981), weiße Headlines.
Jedes Bild zeigt einen anderen Bereich: [BEREICHE]

Direkt als PNG ausgeben, nicht als HTML.
Branding: Produktname als Pill oben, URL unten.
```
