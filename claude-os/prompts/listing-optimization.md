# Prompt: Listing-Optimierung für digitale Produkte

## Verwendung
Wenn du ein neues Produkt auf Etsy oder Gumroad listen willst.

## Prompt
```
Ich verkaufe [PRODUKT] auf [PLATTFORM] für [PREIS].
Zielgruppe: [ZIELGRUPPE]
Hauptfeatures: [FEATURES]

Erstelle mir:
1. Optimierten Titel (max. 140 Zeichen für Etsy)
2. Vollständige Beschreibung mit klarer Struktur
3. 13 relevante Tags (für Etsy)
4. Käufer-Nachricht nach dem Kauf

Preis korrekt: [PREIS] auf [PLATTFORM].
Keine Erwähnung anderer Plattformen in der Beschreibung.
```
