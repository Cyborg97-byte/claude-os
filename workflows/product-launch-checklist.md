# Workflow: Digitales Produkt Launch Checklist

## Gumroad
- [ ] PDF mit Template-Link als Download (kein externer Link)
- [ ] Preis korrekt
- [ ] Beschreibung: kein Verweis auf andere Plattformen
- [ ] 8 Produktbilder hochgeladen
- [ ] Käufer-Nachricht gesetzt (Receipt Tab)

## Etsy
- [ ] Titel: keyword-optimiert, max 140 Zeichen
- [ ] 13 Tags ausgefüllt
- [ ] Beschreibung: plattform-spezifischer Preis
- [ ] 10 Produktbilder
- [ ] Käufer-Nachrichten (allgemein + digital) gesetzt
- [ ] Standort: Deutschland
- [ ] Kategorie korrekt

## Notion Template
- [ ] "Als Vorlage duplizieren" aktiviert
- [ ] Im Inkognito testen: Duplicate-Button sichtbar?
- [ ] User Manual Seite vorhanden
- [ ] Demo-Daten drin

## ProductHunt
- [ ] Account erstellt
- [ ] Upcoming-Profil angelegt
- [ ] Launch-Dienstag festgelegt
- [ ] Tagline, Beschreibung, First Comment vorbereitet
